iPhone / iPad

Use Safari only. Chrome on iPhone cannot add websites to the home screen.
  1. 1 Open this site in Safari (not Chrome).
  2. 2 Tap the Share button □↑ at the bottom of Safari.
  3. 3 Scroll down and tap Add to Home Screen.
  4. 4 Tap Add in the top right.

If you don’t see “Add to Home Screen”, scroll the share sheet further down — it is often below “Add Bookmark”.

Android

  1. 1 Open this page in Chrome (not Facebook/Instagram).
  2. 2 Tap the Chrome menu ⋮ (top right).
  3. 3 Tap Install app or Add to Home screen → Install.

If the blue button does not appear, use Chrome menu ⋮ → Install app. Stay on this page 10–15 seconds first so Chrome can enable install.

Computer

In Chrome or Edge: open the address bar install icon, or menu → Install SupplierAdvisor / Cast, save, and share → Install page as app.
