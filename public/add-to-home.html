iPhone / iPad — Safari only

Chrome on iPhone cannot add to Home Screen. You must use Safari.
  1. Open Safari and this page: supplieradvisor.com/add-to-home.html
  2. Tap Share □↑ (bottom centre of Safari)
  3. Scroll the sheet and tap Add to Home Screen
  4. Tap Add (top right)

Don’t see “Add to Home Screen”? Scroll the share list further — it is often under “Add Bookmark”.

Android — Chrome

  1. Stay on this page in Chrome for 10–15 seconds
  2. Tap Chrome menu ⋮ (top right)
  3. Tap Install app or Add to Home screen
  4. Confirm Install

If menu has no Install yet: use Reset app cache above, reload, wait 15s, open ⋮ again.
